    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

    Different between innerHTML,innerText and textContent

  • innerHTML

    • Returns or sets the HTML content inside an element.

    It can execute scripts and render HTML elements.

    When it is used it modifying structure content but it have a security risk.


  • innerText

    • Returns or sets the visible text inside an element.

    Does not include the hidden element.

    Ignores extra spaces and formatting.


  • textContent

    • Returns or sets all text inside an element, including hidden elements.

    Does not parse HTML.

    Keeps white spaces and formatting.


    Question-3: What is event delegation in the DOM?

    Event Delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, you add a single event listener to a parent element and handle events as they "bubble up" (event bubbling).

    Question-4: What is event bubbling in the DOM?

    Event Bubbling is a concept in JavaScript where an event triggered on a child element bubbles up through its ancestors, triggering the same event listeners on parent elements unless stopped.
